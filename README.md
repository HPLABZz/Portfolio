# Portfolio

🚀 Portfolio Website

A modern, responsive personal portfolio website showcasing my projects, technical skills, and professional background. Built with a focus on performance, clean UI, and mobile-first design, the site highlights full-stack and AI/ML projects with live demos and source code links.


✨ Features

Responsive design optimized for desktop and Android devices

Interactive hero section with animated profile effects

Dedicated projects section with live demos & GitHub links

Smooth scrolling and scroll-based animations

Functional contact form powered by EmailJS

Clean, scalable UI using modern CSS and JavaScript


🛠 Tech Stack

Frontend: HTML5, CSS3, JavaScript

Styling: Custom CSS, animations, gradients

Icons & Fonts: Font Awesome, Google Fonts

Form Handling: EmailJS


📌 Purpose

This portfolio serves as a central hub to present my full-stack development and machine learning projects, demonstrate UI/UX capabilities, and provide an easy way to connect for collaborations or opportunities.
